    Class CallThrottle

    A class providing support for methods that must not be called too often (e.g. UI updates).

    Inheritance
    System.Object

    Syntax
    public sealed class CallThrottle : IDisposable
    Remarks

    Gurantees that a gap of at least X ms will be left between calls to the wrapped method. If this is exceeded then all but the last invocation will be ignored and the last call will be delayed until at least X ms have passed.
